# 大学生制霸生成器

一个记录大学生活副本完成度的网页小工具。

用户可以选择自己经历过哪些大学生活事件，并为每一项设置等级，最后生成制霸率、称号和可分享图片。

## 项目定位

这不是严肃测评，而是一个面向大学生的娱乐型自我表达工具。

## 初版功能

- 显示大学生活项目列表
- 为每个项目选择等级
- 自动计算制霸率
- 根据分数生成称号
- 生成分享图

## 等级设计

- 没体验
- 体验过
- 熟练
- 痛苦精通
- 已经麻木

## 称号设计

- 0%-20%：大学新手村
- 20%-40%：校园观察员
- 40%-60%：普通大学生
- 60%-80%：期末周幸存者
- 80%-95%：大学生活制霸者
- 95%-100%：校园传说
